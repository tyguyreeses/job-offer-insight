## Decision Log Maintenance

Maintain a concise decision history for important choices.

### What to log

1. Schema and validation decisions
2. Architectural boundaries and dependency patterns
3. UI interaction rules with non-obvious rationale
4. Deferred feature rationale
5. Reference adaptation decisions

### Minimum decision record format

1. Decision title
2. Date
3. Context
4. Decision
5. Alternatives considered
6. Consequences

### Completion criteria

Critical project choices are documented and easy to revisit.
